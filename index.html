Horse Incident Reporter

What the App Is

  • A simple mobile-friendly app where users can file a report if they see something wrong with carriage horses.
  • The reports would eventually go to the right groups (ASPCA, city agencies, or others).

Core Features

  1. Incident Categories (Dropdown Menu)
    • Animal distress (overworked, injured, collapsed)
    • Safety violations (traffic issues, illegal parking, blocking crosswalks)
    • Heat/weather concerns
    • General misconduct by carriage drivers
    • Other (with write-in option)
  2. Location & Time
    • Manual entry option (address or nearest landmark)
    • Date + time picker
  3. Description Box
    • Open text field for details
    • Option to include carriage number, horse description, or operator details
  4. Photo Upload
    • User can attach 1–2 photos
  5. Contact Info (Optional)
    • Name, email, or phone — only if user wants to provide
  6. Emergency Notice (See below)
  7. Submit Info
    • Store the report in a simple database or email it

Extra Features (Planned)

  • Report confirmation screen (“Your report has been submitted. Thank you for helping protect NYC horses.”)
  • Tracking number for the report
