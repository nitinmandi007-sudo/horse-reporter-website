Horse Incident Reporter

Horse Incident Reporter is a mobile app that helps users quickly report incidents involving carriage horses.

The app allows you to:

  • Select incident categories (distress, safety, heat, misconduct, etc.)
  • Enter location & time details
  • Add descriptions, carriage/horse details
  • Attach photos
  • Send a report directly via email

This is a community-driven tool and is not affiliated with any official city agency.
